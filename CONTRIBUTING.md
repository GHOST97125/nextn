# 🤝 Guide de Contribution – InspectSignal

Merci de votre intérêt pour contribuer à InspectSignal ! Ce guide vous aidera à participer efficacement au développement du projet.

---

## ✅ Pré-requis

Avant de commencer, assurez-vous d’avoir :

- Android Studio installé
- Un compte Firebase configuré
- Les connaissances de base en Kotlin, XML et Firebase
- Node.js et npm (si applicable pour Firebase Studio)
- Un fork du dépôt GitHub

---

## 🧪 Bonnes pratiques

- Respecter la structure du projet
- Écrire un code lisible et documenté
- Ajouter des commentaires si nécessaire
- Tester vos modifications avant de proposer une PR
- Utiliser des noms de branches explicites (`feature/`, `fix/`, `refactor/`)

---

## 📥 Étapes pour contribuer

1. **Forkez** le dépôt :
   Cliquez sur "Fork" en haut à droite de la page GitHub.

2. **Clonez** votre fork :
   ```bash
   git clone https://github.com/votre-utilisateur/InspectSignal.git
   ```
