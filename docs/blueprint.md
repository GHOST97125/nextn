# **App Name**: Signal Inspector

## Core Features:

- Image Upload: Upload photos of road signs to be analyzed.
- Image Gallery: Display uploaded images in a gallery with input fields for road sign IDs.
- Image Analysis: Use AI to analyze the images and identify defects. The AI will reason about whether each sign meets guidelines.
- Map Display: Display a map with markers indicating the location of each road sign based on EXIF data.
- Report Generation: Summarize the analysis results and display them in a report.
- Action Planning: Use AI to generate a list of materials needed for the repairs, and suggest a set of safety guidelines.
- Copy to Clipboard: Copy the generated report to the clipboard.

## Style Guidelines:

- Primary color: Teal (#2EC4B6) to represent safety and visibility.
- Background color: Light gray (#F1F5F9) to provide a neutral backdrop.
- Accent color: Orange (#FF9F1C) to highlight interactive elements and call to action; provides a visual cue and evokes a sense of urgency where needed.
- Font: 'Inter' (sans-serif) for both body and headlines to provide a modern and readable interface.
- Use simple, clear icons to represent different types of road signs and potential defects.
- Use a clean, minimalist layout to focus on the images and analysis results.
- Use subtle animations to provide feedback when the AI is processing images and generating the report.
