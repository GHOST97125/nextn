# Firebase Studio – InspectSignal

InspectSignal est une application Android développée avec Firebase Studio. Elle permet l'inspection de signaux avec des fonctionnalités avancées telles que la synchronisation Firebase, une base SQLite locale, une visionneuse PDF, et une IA embarquée.

![Snyk Security](https://snyk.io/test/github/baltimoreulrich8/nextn)

## 🚀 Pour commencer

Consultez le fichier `src/app/page.tsx` pour démarrer avec le projet.

## 🔧 Fonctionnalités principales

- Authentification Firebase
- Base de données SQLite locale
- Synchronisation avec Firestore
- Visionneuse PDF intégrée
- Intelligence Artificielle embarquée
- Fonctionnement en ligne et hors ligne
